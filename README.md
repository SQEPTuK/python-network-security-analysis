## Описание проекта

Данный проект представляет собой простой инструмент для анализа событий
сетевой безопасности.\
Скрипт обрабатывает логи системы обнаружения вторжений Suricata и
выявляет IP‑адреса, которые чаще всего встречаются в подозрительных
событиях.

На основе полученных данных формируется сводный отчет и строится график
активности.

## Используемые технологии

-   Python
-   pandas
-   matplotlib
-   requests

## Входные данные

Для анализа используется файл логов Suricata:

suricata_eve.jsonl

Файл содержит события сетевой активности в формате JSON.

## Результаты работы программы

После выполнения скрипта формируются следующие файлы:

-   **report.csv** --- таблица с количеством подозрительных событий по
    IP‑адресам\
-   **report.json** --- отчет в формате JSON\
-   **top_suspicious_ips.png** --- график с наиболее активными
    IP‑адресами

Все результаты сохраняются в папке `output`.
